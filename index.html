What is it?

The dropZone

The dropZone, or dropZones, are two "hot" folders where users drop files (jpg, excel, word, etc.) with a specific naming standard, that are then ingested via a script into the Multimedia module.

Naming Standard for Catalog Module

  • creator= LastName, FirstName
  • resource type= Museum for Publication
  • object number= 3-21-1986
  • sequence number= 01

User drops file(s) into folder

↓↓↓

Script begins to run

Query Multimeida module for new records

The script strips the creator and resource type from the file name, so that the file (identifier) becomes the object number and sequence number. The creator and resource type from the file are inserted into the Creator and Resource Type field

A list view is used so users can quickly review.
